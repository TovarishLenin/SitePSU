# Local Landing Framework

Базовый локальный проект под лендинг из 7 секций.

## Структура

- `index.html` — основная структура страницы
- `styles.css` — базовые стили и адаптив
- `script.js` — меню и анимация появления секций
- `docs/site-wireframe.pdf` — текущий PDF-макет из Figma
- `assets/img/site-wireframe-preview.png` — PNG-превью макета для быстрой сверки
- `assets/img/` — изображения для секций
- `assets/icons/` — иконки и SVG
- `assets/fonts/` — локальные шрифты, если понадобятся
- `sections/` — отдельные HTML-заготовки под секции макета
- `docs/` — дополнительные PDF, описания и референсы
